In addition, if the declarations differ because one type is an incomplete structure, union or enumeration type and the other is the corresponding completed type with the same tag, the types are taken to agree. Moreover if one type is an incomplete array type and the other is a completed array type, the types, if otherwise identical, are also taken to agree. Finally, if one type specifies an old-style function and the other an otherwise identical new-style function with parameter declarations, the types are taken to agree.
