In the old-style declarator, the identifier list must be absent unless the declarator is used in the head of a function definition. No information about the types of the parameter is supplied by the declaration.


int f(), *fpi(), (*pfi)();


The function 'f' returns an integer, the function 'fpi' returns a pointer to an integer and a pointer 'pfi' points to a function returning an integer. In none of these are the parameter types specified, they are old-style.
